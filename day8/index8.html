Strings

String is a Collection of Characters Enclosed with a Single qoutes or Double qoutes

String Length

    EX :  let myStr = "farhan";
    let len = myStr.length;
    console.log(len); 
                            // o/p : 9

Concatination

It concat two string.

     var t1="farhan";
     var t2="Md";
     var t3=t1.concat(" ",t2);
     console.log(t3);         //o/p: farhan Md

Trim

It removes unnecessary spaces.

    let str="           malaysiaa";
    console.log(str.len);
    let tr = str.trim(); //20
    console.log(tr);        // malaysiaa
    console.log(tr.len);         //9

Reverse And Join

It reverse the string & It converts array to string.

    let rev1 = myStr.split("").reverse();
    console.log(rev1);  // ['a','e','i','o','u']
    let rev2 = myStr.split("").reverse().join("");
    console.log(rev2);              // o/p: aeiou
